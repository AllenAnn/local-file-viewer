# 欢迎使用本地文档预览工具！ 🎉

这是一个专门为您构建的本地文档预览小工具。

### 支持的文件类型：
1. **Markdown (`.md`)**：支持完整的 Markdown 语法，包括列表、代码块、引用等。
2. **Word 文档 (`.docx`)**：高保真地渲染 Word 格式、段落、表格。
3. **Excel 表格 (`.xlsx`, `.xls`, `.csv`)**：支持解析多工作表表格，提供现代化的网格展示，完美支持大表格滚动。

### 使用方法：
* **第一步**：直接将您的 `.md`、`.docx`、`.xlsx`、`.xls`、`.csv` 文件放入当前目录下的 `files/` 文件夹中。
* **第二步**：如果您新增或删除了文件，只需点击左侧文件列表下方的 **“刷新列表”** 按钮即可，网页会自动同步！
