# 本地文档预览工具 (Local File Viewer)

这是一个极为轻量、美观的本地文档管理与在线高清预览小工具。

## ✨ 特性

- ⚡ **零安装依赖**：后端使用原生 Node.js，无需运行 `npm install` 下载任何依赖包，解压即用。
- 🎨 **现代化 UI 设计**：支持毛玻璃效果、响应式布局、以及明亮/暗黑主题的一键切换。
- 📦 **免安装 Office**：基于强大的前端 JavaScript 库，在浏览器端直接完成全部渲染，无需安装 Office 软件。
- 🔍 **便捷检索**：支持对文件名进行实时模糊检索与过滤。
- 🔄 **热刷新机制**：在 `files/` 目录下放置、删除文件后，网页中点击“刷新文件列表”即可实现即时同步。

## 📂 支持的预览格式

1. **Markdown (`.md`)**：完美解析各级标题、代码块、列表、表格和引用。
2. **Word 文档 (`.docx`)**：高保真模拟 A4 纸张页面渲染，包括文档样式、表格和字体。
3. **Excel 表格 (`.xlsx`, `.xls`, `.csv`)**：提供现代化的网格形式展现，支持多工作表 (Sheet) 交互切换。

## 🚀 快速使用

1. **放置文件**：将您需要查看的任意 `.md`, `.docx`, `.xlsx`, `.xls` 或 `.csv` 文档直接拖入当前文件夹下的 **`files/`** 目录中。
2. **启动程序**：在 Windows 系统中，直接双击运行同级目录下的 **`start.bat`** 批处理文件。
3. **浏览查看**：服务启动后，系统会自动在您的默认浏览器中打开页面：`http://localhost:8989`，即可在侧边栏点击进行预览。

## 🛠️ 后台运行说明

服务默认运行在 `8989` 端口，若希望在其他机器或手机访问本电脑的文件：
* 保证本电脑和另一台设备在**同一个局域网内**。
* 另一台设备在浏览器中输入 `http://<您的局域网IP>:8989` 即可访问。
